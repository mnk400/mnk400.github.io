Projects I’ve recently worked on

Rocketnani

A simple program written in C for terminal emulators, which flies a rocket in the terminal window.

Weather Station

An IOT implementation of a weather station created using an Arduino and a Raspberry pi with a few sensors. The data was displayed on a webpage and tweeted to a tweeter account every 5 minutes.

Link aggregator

A barebones link aggregator created using ruby on rails, the website had basic functionality of the website reddit.

Compile

A sort of make alternative created for personal use, which makes compiling multiple files at once really easy for me. This project is currently closed source but will be released when it’s stable enough.
