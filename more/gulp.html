A native macOS front-end for gallery-dl with history & log management, built with SwiftUI.

Gulp is a simple tool that helps you download images and videos from various websites by providing an easy to use UI around the gallery-dl command line tool.

The app also keeps track of your download history so you can revisit past downloads and past logs. Out of the box, gulp supports all platforms that gallery-dl does.

Download for MacOS 26+ brew install --cask gulp
