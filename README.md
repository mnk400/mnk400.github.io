# Personal Portfolio & Blog

This is an Astro-based personal website that serves as a portfolio, blog, photo archive, and showcase for interactive projects.

## Development

1. Install dependencies

```bash
npm install
```

2. Start the development server

```bash
npm run dev
```

3. Build the static site

```bash
npm run build
```

## Structure

- `src/pages/` contains routes.
- `src/components/` contains reusable components.
- `src/content/posts/` contains blog posts.
- `src/data/more/` contains typed metadata for the `/more/` index and project/archive pages.
- `src/styles/` contains Sass partials and the main stylesheet entry.
- `public/assets/` contains static images and data served at `/assets/*`.
